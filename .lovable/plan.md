

# الحل المباشر: تعطيل refetchOnWindowFocus

## المشكلة بإختصار
- React Query يعيد تحميل كل البيانات عند العودة للتطبيق من تبويب آخر
- إذا فشل الاتصال في تلك اللحظة → تختفي المنظمة وتظهر "Personal Account"

## الحل (تعديل بسيط واحد)

### الملف: `src/App.tsx` - السطر 61

**قبل:**
```typescript
const queryClient = new QueryClient();
```

**بعد:**
```typescript
const queryClient = new QueryClient({
  defaultOptions: {
    queries: {
      refetchOnWindowFocus: false,
    },
  },
});
```

## ماذا يفعل هذا؟
| السلوك | قبل | بعد |
|--------|-----|-----|
| العودة من YouTube | إعادة تحميل كل شيء | لا شيء يحدث |
| فشل الاتصال عند العودة | تختفي البيانات | البيانات تبقى |
| الانتظار 30 ثانية | يحدث | لا يحدث |

## ملاحظة
هذا لن يؤثر على:
- تسجيل الدخول والخروج (يعمل طبيعياً)
- إضافة بيانات جديدة (تُحدّث طبيعياً)
- الـ Real-time updates (تعمل طبيعياً)

فقط يوقف "الإعادة التلقائية المزعجة" عند الرجوع للصفحة.
